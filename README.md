# 🌿 Plant Species Classification using CNN + SVM + KNN

## 📌 Overview

A hybrid deep learning project developed as part of the B.Tech Mini Project at SASTRA Deemed University.

The system combines Convolutional Neural Networks (CNN) with traditional machine learning classifiers such as Support Vector Machine (SVM) and K-Nearest Neighbors (KNN) to improve plant species classification accuracy from leaf images.

## 🚀 Features

- Automatic plant species identification from leaf images
- CNN-based feature extraction
- Hybrid CNN + SVM classification
- Hybrid CNN + KNN classification
- Data preprocessing and augmentation
- Model performance evaluation using multiple metrics

## 🛠️ Technologies Used

- Python
- TensorFlow / Keras
- Scikit-learn
- NumPy
- Pandas
- Matplotlib

## 📂 Datasets

- Swedish Leaf Dataset
- LeafSnap Dataset
- Flavia Dataset
- MalayaKew Dataset

## 📊 Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix
